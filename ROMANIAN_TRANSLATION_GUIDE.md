## Implementation Notes

### HTML Files
Replace all English text in:
- Page titles (`<title>` tags)
- Headers (`<h1>`, `<h2>`, etc.)
- Labels (`<label>` tags)
- Button text (`<button>` content)
- Link text (`<a>` content)
- Placeholder text (`placeholder` attributes)
- Modal content

### JavaScript Files
Replace all English strings in:
- Alert messages
- Confirm dialogs
- Success messages
- Error messages
- Dynamic content generation
- Table headers created dynamically
- Form validation messages

### Consistency Guidelines
1. Use formal "you" (dumneavoastră) in user messages
2. Use consistent terminology throughout
3. Keep technical terms in English where appropriate (e.g., "PDF", "ID")
4. Maintain button case consistency (capitalize first letter)
5. Use Romanian-specific characters: ă, â, î, ș, ț
